Detailed Description

Class for creating a 3D linearized two-node frame element in a mesh.

See also
Element.hpp Mesh.hpp

Creates a lin3DFrame2 in a finite element Mesh.

Parameters
nodesThe Node connectivity array of this Element.
sectionPointer to the Section that this Element is made out of.
formulationThe frame formulation: "Bernoulli", "Timoshenko".
quadratureThe integration rule to be employed.
nGaussNumber of Gauss points for Element integration.
Note
More details can be found at lin3DFrame2.

lin3DFrame2::theNodes, lin3DFrame2::theSection, lin3DFrame2::QuadraturePoints.
