Static point-load

Represent a vector that is applied on a Node object. This force is represented by a magnitude and a direction. The Load object created in this manner is assembled directly to the global force vector, thus its direction must coincide with the global coordinate system.

An static point Load is created using the built-in parse-structure provided in the Parser,

POINTLOAD Tag STATIC value nd dir nn nodes

Variable Description
Tag Unique load object identifier.
value Magnitude/intensity of the applied force, traction or body load.
nd Number of dimension for the Load.
dir Vector containing the direction of the force.
nn Number of nodes where this Load will be applied.
nodes List of Node identifiers that share this load.
Attention
The number of dimensions nd is the number of degree-of-freedom of that Node for POINTLOAD.

Example

A 3D static point load applied downwards on node 2 whose intensities is 0.75 N:
POINTLOAD 1 STATIC 0.75 3 0.000 0.000 -1.000 1 2
