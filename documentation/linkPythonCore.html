Core

The python Core folder has the following Python files:

  • Definitions.py
  • Numberer.py
  • Partition.py
  • PlaneWave.py
  • RandomField.py
  • Utilities.py
  • Outputs.py
  • SeismoVLAB.py

The Core folder provided with two main python files: SeismoVLAB.py that loads all required modules to generate a model; and Definitions.py that contains the data structure (dictionaries) that stores the finite element information. Also, Python routines provided in SVL handles the degree-of-freedom numbering in Core/Numberer.py, the domain partition in Partition.py, soil spatial variability in RandomField.py, and domain reduction forces in PlaneWave.py. Other features can be incorporated to meet the user's need as well.
