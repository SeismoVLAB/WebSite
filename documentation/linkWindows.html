Windows Operating System

The Windows installation requires of a C++ compiler. We recommend to install MSYS2 (Download from https://www.msys2.org/) C/C++/make. Follow for instance a tutorial on youtube (https://www.youtube.com/watch?v=sMbPdSx5i8o). In addition to the directory C:\msys64\mingw64\bin mentioned in the above video, also add C:\msys64\usr\bin to the variable named Path in Environment Variables. The procedure is identical. Next, make sure these two directories have priority (are on top of the list) to all other directories otherwisethe system will confuse similar Linux and Windows commands (such as find) and compilation will fail. Then, download Eigen C++ library, MUMPS Library, and Pestc Library.

  • The Eigen C++ library can be downloaded from the website http://eigen.tuxfamily.org/. This package needs to be unzip and its content move (for instance) to
    C:\msys64\usr\include\Eigen.
  • The MUMPS library can be downloaded from the website http://mumps.enseeiht.fr/. This package needs to be unzip and compiled (for instance) at
    C:\msys64\usr\include\Mumps.
  • The Pestc Library library can be downloaded at the website https://www.mcs.anl.gov/petsc/. This package needs to be unzip and compiled (for instance) at
    C:\msys64\usr\include\Petsc.

Modify the Makefile.mk file such the previous path point to the right libraries:

EIGEN_DIR = C:\msys64\usr\include\Eigen
PETSC_DIR = C:\msys64\usr\include\Petsc
MUMPS_DIR = C:\msys64\usr\include\Mumps

Also, check that : MPI_DIR, METIS_DIR, SCOTCH_DIR have the correct path where these libraries are installed. Make sure libraries such as: -lscalapack-openmpi -lblacs-openmpi -llapack -lblas and -lparmetis -lmetis -lptesmumps -lptscotch -lptscotcherr are also installed.

Finally, write in terminal:

make -s DEBUG=False
