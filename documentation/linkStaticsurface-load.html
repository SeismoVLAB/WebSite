Static surface-load

Represent a traction that is applied on an Element object. This force is represented by a magnitude and direction that is applied on a face of the element. The Load object created in this manner needs first to be transformed at the element level into nodal forces and then assembled directly to the global force vector. The traction load direction needs to coincide with the global coordinate system.

An static surface Load is created using the built-in parse-structure provided in the Parse,

ELEMENTLOAD Tag STATIC SURFACE value nd dir ne elems faces

Variable Description
Tag Unique load object identifier.
value The applied surface load magnitude.
nd The number of dimension of this Analysis.
dir The unit vector of the applied surface force.
ne Number of elements where this Load will be applied.
elems List of Element identifiers that share this load.
faces List of Element faces identifiers where the traction is applied.
Attention
The direction of the load must coincide with the global coordinate system.
For Shell elements only out-of-plane tractions can be applied. The out-of-plane face number is 5.
The face number is provided according to the element connectivity. See Figure bellow for details of face number for line, quadrilateral, and hexahedral elements.

Example

A 3D static surface load applied downwards on elements 1,2,3,4,5 on face 6 with magnitude 50.0 N/m^2:
ELEMENTLOAD 1 STATIC SURFACE 50.00 3 0.000 0.000 -1.000 5 1 2 3 4 5 6 6 6 6 6
