Display.py

The python Display.py file is a set of functions that allow to generate a 3D visualization.

  • renderData(option):
    This function render a 3D view of the model created so far. The elements in the mesh are colored depending if option='element' or option='partition'. The default color for each option are given at Definitions.py (VTKcolors dictionary)
  • setVTKtype(option):
    This function classifies each element (depending on its VTKname defined in Definitions.py at VTKelems dictionary) and associate a colordepending if option='element' or option='partition'.
Attention
The python3 module VTK must be installed to use the Display.py functions
