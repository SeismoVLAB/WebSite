Dynamic point-load

Represent a vector that is applied on a Node object. This force is represented by a magnitude that changes in time and direction. The Load object created in this manner is assembled directly to the global force vector, thus its direction must coincide with the global coordinate system.

A dynamic point Load is created using the built-in parse-structure provided in the Parser,

POINTLOAD Tag DYNAMIC path nd dir nn nodes

Variable Description
Tag Unique load object identifier.
path Represents the local address where the file is located.
nd Number of dimension for the Load.
dir Vector containing the direction of the force.
nn Number of nodes where this Load will be applied.
nodes List of Node identifiers that share this load.
Attention
The file located at path for dynamic analysis contains the number of time steps, and then a vector with the varying intensities in time. The time step of such intensities is assumed to coincide with the time step of the simulation.

Example

A 2D dynamic point load applied downwards on node 2 whose intensities are located at ../Exterior.txt:
POINTLOAD 1 DYNAMIC ../Exterior.txt 2 0.000 -1.000 1 2
