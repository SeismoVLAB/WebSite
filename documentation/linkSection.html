Section

The Section keyword defines a cross-section to be assigned in an structural element. This object provides member functions to compute the section's geometric properties as well as internal forces developed in the section. The implementation of each section is provided in each derived-class. A Section object is meant to be used in frame elements such as lin2DFrame2, kin2DFrame2, lin3DFrame2, and kin3DFrame2, as well as shell elements like lin3DShell4. The name of each section derived-class has the following structure:

Formulation-Dimension-Name.hpp

.

Name Description
Formulation Stands for either Linearized (lin), or Fiber (fiber).
Dimension One-dimensional (1D), two-dimensional (2D), or three-dimensional (3D).
Name The name of the section.

The following sections are available in SVL

  • Lin3DRectangular
  • Lin3DAngle
  • Lin3DChannel
  • Lin3DTee
  • Lin3DWideFlange
  • Lin3DCircular
  • Lin3DRectangularTube
  • Lin3DCircularTube
  • Lin3DThinArea
  • Local Axes
  • Insertion Point
