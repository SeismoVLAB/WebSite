CompositeBathe

The CompositeBathe keyword creates a two-step implicit second order accurate integrator class that is used for dynamic analysis. First step assumes trapezoidal rule, this is displacement and velocity are approximated as,

\[ V_{n+{^1/{_2}}} = V_{n} + \frac{\Delta t}{4} \left( A_{n+{^1/{_2}}} + A_n \right) \nonumber \\ U_{n+{^1/{_2}}} = U_{n} + \frac{\Delta t}{4} \left( V_{n+{^1/{_2}}} + V_n \right) \nonumber \]

thus, the effective stiffness matrix and force vector are computed as,

\[ \textbf{K}_{\textrm{eff}} = \textbf{K}^{(i-1)}_{n+{^1/{_2}}} + \frac{16}{\Delta t^2} \textbf{M} + \frac{4}{\Delta t} \textbf{C} \nonumber \\ \textrm{F}_{\textrm{eff}} = \rm{R}_{n+{^1/{_2}}}^{(i)} - \textrm{F}_{n+{^1/{_2}}}^{(i-1)} + \textbf{M} \left(\frac{8}{\Delta t}V_n + A_n - \frac{16}{\Delta t^2} \Delta U \right) + \textbf{C} \left(V_n - \frac{4}{\Delta t} \Delta U\right) \nonumber \]

Second step assumes a 3-point Euler backward method, this is velocity and acceleration are approximated as,

\[ V_{n+1} = \frac{1}{\Delta t} \left( U_{n} - 4\, U_{n+{^1/{_2}}} + 3\, U_{n+1}\right) \nonumber \\ A_{n+1} = \frac{1}{\Delta t} \left( V_{n} - 4\, V_{n+{^1/{_2}}} + 3\, V_{n+1}\right) \nonumber \]

thus, the effective stiffness matrix and force vector are now computed as,

\[ \textbf{K}_{\textrm{eff}} = \textbf{K}^{(i-1)}_{n+1} + \frac{9}{\Delta t^2} \textbf{M} + \frac{3}{\Delta t} \textbf{C} \nonumber \\ \textrm{F}_{\rm{eff}} = \textrm{R}_{n+1}^{(i)} - \textrm{F}_{n+1}^{(i-1)} + \textbf{C} \left( \frac{3}{\Delta t}U_{n+1}^{(i-1)} - \frac{4}{\Delta t}U_{n+{^1/{_2}}} + \frac{1}{\Delta t}U_{n}\right) - \textbf{M} \left(\frac{9}{\Delta t^2}U_{n+1}^{(i-1)} - \frac{12}{\Delta t^2}U_{n+{^1/{_2}}} + \frac{3}{\Delta t^2}U_{n} - \frac{4}{\Delta t}V_{n+{^1/{_2}}} + \frac{1}{\Delta t}V_{n}\right) \nonumber \]

A CompositeBathe integrator can be created using the built-in parse-structure provided in the Parser as,

INTEGRATOR COMPOSITEBATHE mTol kTol fTol dt

Variable Description
mTol The specified tolerance for setting \(M_{ij} \sim 0.0\).
kTol The specified tolerance for setting \(K_{ij} \sim 0.0\).
fTol The specified tolerance for setting \(F_{j} \sim 0.0\).
dt The time step for the analysis.
Attention
The default tolerance for \(M_{ij}\), \(K_{ij}\), and \(F_{j}\) is \(10^{-15}\).
