Method

The python Method folder has the following Python files:

  • Attach.py
  • Remove.py
  • Builder.py
  • Display.py
  • Compute.py

The Method folder provided with the python scripts Attach.py and Remove.py that allows to add/remove Entities in SVL.Entities[...] dictionary. It also provides with builder.py to generate simple domains in 1D, 2D, and 3D and discretize it. Similarly, several domains can be merged to generate more complex models, and they can be display using the VTK functionalities using Display.py.
