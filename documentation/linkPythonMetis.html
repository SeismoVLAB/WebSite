Metis

The Metis folder has a compiled version of the Metis software for different platforms (Linux, MacOS, and Windows). The Metis software is employed to generate the domain decomposition which is employed in Seismo-VLAB to perform a parallel execution. Basically, the model domain (i.e., group of objects such as Node, Element, Material, Section, and Load) is divided so that the number of elements are almost uniform across processors. Metis is called from the function SetMetisInputFile() defines in Partition.py.

Note
The latest Metis version can be downloaded at http://glaros.dtc.umn.edu/gkhome/metis/metis/overview; however this version may require to be compiled.
The address where metis program is downloaded must be consistent with: 
SVL.Options['metispath']
described in Definitions.py. By default, this path is set to be ./SVL/01-Pre_Process/Metis/mpmetis
