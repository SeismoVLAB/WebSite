Integrator

The Integrator keyword specifies how the StaticAnalysis or DynamicAnalysis is performed. This abstract class have member functions that uses the Assembler class to compute the effective stiffness matrix \(\textbf{K}_{\textrm{eff}}\) and effective force vector \(\textrm{F}_{\textrm{eff}}\) to be provided to the LinearSystem object to compute the displacements \(U\), velocities \(V\) and acceleration \(A\) vector states for the incremental analysis. In the incremental analysis, at each (pseudo) time step \(t_n\), one needs to solve nonlinear systems of equations iteratively until satisfying the given tolerance and convergence criterion.

The available integrators in Seismo-VLAB are:

  • QuasiStatic
  • CentralDifference
  • NewmarkBeta
  • CompositeBathe
