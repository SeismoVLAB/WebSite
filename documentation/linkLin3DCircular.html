Lin3DCircular

The CIRCULAR keyword creates a line Section that is represented by its height \(h\), width \(b\), flange thickness \(t_f\), and web thickness \(t_w\). This class computes the cross-sectional properties such as area, shear area, inertias and so on. The figure shows a representation of the Lin2DCircular and Lin3DCircular

The section properties are:

  • Area properties: \(A = \pi \, r^2 \,, \; As_2 = \frac{9}{10} A \,, \; As_3 = \frac{9}{10} A \)
  • Inertia properties: \(I_{11} = \frac{\pi}{2} r^4 \,, \; I_{22} = \frac{\pi}{4} r^4 \,, \; I_{33} = \frac{\pi}{4} r^4\)
  • Geometric center from bottom left corner: \(x_3 = r\), and \(x_2 = r\)

A CIRCULAR Section object can be created using the built-in parse-function provided in the Parse as follows,

SECTION PLAIN Tag LINE mTag CIRCULAR r

Variable Description
Tag Unique Section object identifier.
mTag Material object identifier.
r The radius of the cross section.
Attention
The CIRCULAR Section object is made of one Material.
The CIRCULAR Section is assumed to be use in linearized and finite deformations.
The strees/strain coordinates at Section::GetStrainAt() or Section::GetStressAt() must be refered to the geometric center \(\bar{x}_3,\bar{x}_2\).
If strees/strain are evaluated outside the section domain, the response is computed at \(x_3 = 0, x_2 = r\) from the axes shown in figure.
