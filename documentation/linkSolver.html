Linear System

The LinearSystem keyword creates an abstract object that solves the system of equations provided with \(\textbf{K}_{\textrm{eff}}\), and \(\textrm{F}_{\textrm{eff}}\) that comes from the Integrator. This abstract class control if a direct/iterative method of solution will be employed for solving the linear system.

The available solvers in Seismo-VLAB are:

  • EigenSolver (serial)
  • MumpsSolver (parallel)
  • PetscSolver (parallel)
Attention
Direct solvers are recommended for small to medium size model ( \(\leq 500 000\) degree-of-freedom) using pararllel computing. These solvers work even better if the Cholesky decomposition is computed once, situation that occurs in linear problems.
Iterative solvers such as Conjugate Gradient will be available in future. Iterative solvers are recommended for large problems ( \(\gg 500 000\) degree-of-freedom) using pararllel computing. These solvers trade accuracy for speed, thus for nonlinear problem the tolerance must be small.
