Compute.py

The python Compute.py file transform a rigid link, diaphragm, and rigid body fields into SVL constraints. It also provides with a fucntion to compute the face number of an element. The following functions are currently available

  • SurfaceFace(name, surf):
    This function associate the face (surface) number to an element (name) depending on the connectivity array spacified for that face (surf).
  • RigidLinkConstraints():
    This function generates the constraints associated to the applied rigid links. The rigid link constraints are available for 2D and 3D. See Attach.py (addRigidLink) for more details.
  • DiaphragmConstraints():
    This function generates the constraints associated to the applied rigid diaphragm. The diaphragm constraints are available for 2D and 3D and needs to be specified along an axis. See Attach.py (addDiaphragm) for more details.
  • RigidBodyConstraints():
    This function generates the constraints associated to the applied rigid body. The rigid body constraints are available only for 3D. See Attach.py (addRigidBody) for more details.
  • ApplyConstraints():
    This function enforces all constarints defines as rigid link, diaphragm, rigid body at the same time.

Example

The diaphragm constraints are enforced using the python interface as follows:
SVL.DiaphragmConstraints()

All special constraints such as rigid link, diaphragm, rigid body can be enforced at the same time as follows:
SVL.ApplyConstraints()
