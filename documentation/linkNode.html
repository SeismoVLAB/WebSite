Node

The Node class defines a point in a finite element mesh. This object provides with member functions to obtain the coordinates Node::Coordinates of the node and the numbering for total Node::TotalDegreeOfFreedom and free degrees of freedom Node::FreeDegreeOfFreedom. This class also stores displacements Node::Displacements, velocities Node::Velocities and accelerations Node::Accelerations at each degree of freedom. The Figure bellow shows a schematic representation of a node in 1-dimension, 2-dimensions and 3-dimensions, respectively. The single arrows represent the translational degree-of-freedom, while the triple arrows represent the rotational degree-of-freedom.

A Node is created using the built-in json parse-structure provided in the Parser.hpp, 

{
    "Nodes": {
        "Tag": {
            "ndof" : int,
            "totaldof": [ ],
            "freedof" : [ ],
            "coords"  : [ ]
        }
    }
}
Variable Description
Tag A unique node number identifier.
ndof Number of degree-of-freedom in the node.
totaldofList of total degree-of-freedom identifiers.
freedof List of free degree-of-freedom identifiers, ( \(= -1\)) is restrain, ( \(\leq -2\)) is constraint, and and ( \(\geq 0\)) is free.
coords The coordinates in 1-, 2-, and 3-dimensions of the node.
Attention
The node identifier number must be unique and positive ( \(\geq0\)).
The numbering in totaldof and freedof defines the sparse matrix pattern.
The node object handles the domain-reduction motion used in Load.
It is worth mentioning that the number of degree of freedom ndof at each node is defined by the user, and such number should be consistent with the element that will be connected to that node. Defining the number of degree of freedom in this manner allows to add explictly internal degree of freedom that can be used to formulate elements, see for instance PML2DQuad4.

Example

A fixed tri-dimensional node with 3 degree of freedom and coordinate (0.0, 0.0, 10.5)
{ "Nodes": { "4": { "ndof" : 3, "totaldof": [12, 13, 14], "freedof" : [-1, -1, -1], "coords" : [0.0, 0.00, 10.50] } } }

A free two-dimensional node with 3 degree of freedom and coordinate (0.20, 0.50)
{ "Nodes": { "48": { "ndof" : 3, "totaldof": [345, 346, 347], "freedof" : [327, 328, 329], "coords" : [0.20, 0.50] } } }
