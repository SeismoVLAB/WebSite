SeismoVLAB.py

This file check the correctness of the defined model and generate the Run-Analysis input files. The functions declared in this file are:

  • Entities2Processor():
    This function creates a dictionary that holds all information required to be written in the k-th processor
  • createPartitions():
    This function creates the partitions according with the pattern generated during the domain decomposition. Basically, goes over the Entities and extract the information
  • checkWarnings():
    This function checks consistency between the parameters specified by the user and the parameters employed in SeismoVLAB. A small report is printed showing all possible warnings encounter during the process.
