Load

The Load class defines a forces or traction that can be applied to either a Node or to an Element respectively. The Load class is designed to:

  • Have an internal structure suitable for linear and nonlinear behavior.
  • Provide with functionalities that allow to construct mass, damping, and stiffness matrices as well as force vector.

The main member function in Load are:

  • Load::GetNodes()
    This function returns the Node list where this Load is applied.
  • Load::GetElements()
    This function returns the Element list where this Load is applied.
  • Load::GetLoadVector()
    This function returns the load force at a certain time step.
  • Load::GetClassification()
    This function returns an integer associated with the load model. So far, these are available:
    • 1 : Static point load.
    • 2 : Dynamic point load.
    • 3 : Static surface load.
    • 4 : Static body load.
    • 5 : Dynamic body load.
    • 7 : Dynamic user's define wave load.
    • 8 : Static/Dynamic displacement support motion.

The load formulation that can be considered in Seismo-VLAB are:

  • Static point-load
  • Dynamic point-load
  • Static surface-load
  • Static body-load
  • Dynamic body-load
  • Dynamic user-defined wave load
  • Static/Dynamic displacement support motion
