Lin3DWideFlange

The LIN3DWIDEFLANGE (or LIN2DWIDEFLANGE) keyword creates a line Section that is represented by its height \(h\), width \(b\), flange thickness \(t_f\), and web thickness \(t_w\). This class computes the cross-sectional properties such as area, shear area, inertias and so on. The figure shows a representation of the Lin2DWideFlange and Lin3DWideFlange

The section properties are:

  • Area properties: \(A = 2 \, t_f \, b + t_w (h - 2 \, t_f) \,, \; As_2 = h \, t_w \,, \; As_3 = \frac{5}{3} b \, t_f\)
  • Inertia properties: \(I_{11} = I_{22} + I_{33}\,, \; I_{22} = \frac{1}{12} \left( 2 \, t_f \, b^3 + (h - 2 \, t_f) \, t_w^3 \right)\,, \; I_{33} = \frac{1}{12} \left( b \, h^3 - (b - t_w)(h - 2\,t_f)^3 \right)\)
  • Geometric center from bottom left corner: \(x_3 = \frac{b}{2} \), and \(x_2 = \frac{h}{2} \)

A LIN3DWIDEFLANGE Section object can be created using the built-in parse-function provided in the Parse as follows,

SECTION Tag PLAIN mTag LIN3DWIDEFLANGE h b tf tw

Variable Description
Tag Unique Section object identifier.
mTag Material object identifier.
h The height of the cross section.
b The width of the cross section.
tf The flange thickness of the cross section.
tw The web thickness of the cross section.
Attention
The LIN3DWIDEFLANGE / LIN2DWIDEFLANGE Section object is made of one Material.
The LIN3DWIDEFLANGE / LIN2DWIDEFLANGE Section is assumed to be use in linearized and finite deformations.
The stress/strain coordinates at Section::GetStrainAt() or Section::GetStressAt() must be referred to the geometric center \(\bar{x}_3,\bar{x}_2\).
If stress/strain are evaluated outside the section domain, the response is computed at \(x_3 = -\frac{b}{2}, x_2 = \frac{h}{2}\) from the axes shown in figure.
